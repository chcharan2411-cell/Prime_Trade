# Prime Trade

A Full Stack Product Management System built using:

- Spring Boot
- Spring Security
- JWT Authentication
- React
- MySQL
- REST APIs

## Features

✔ User Authentication
✔ JWT Security
✔ Role-Based Access
✔ Product CRUD
✔ Search
✔ Pagination

## Project Structure

Prime_Trade/
├── backend/
└── frontend/

## Run Backend

cd backend
mvn spring-boot:run

## Run Frontend

cd frontend
npm install
npm run dev
